# 2Dシューティングゲーム集

4つの異なるアプローチで実装された2Dシューティングゲームのコレクションです。ゲーム開発の学習用に、それぞれ異なる特徴と実装方法を持たせています。

## ゲーム一覧

### 1. クラシック版 (`game1_classic`)
**難易度: 基本**

シンプルで分かりやすい基本的な2Dシューティングゲーム。プレイヤーが弾を撃って敵を倒す、ゲーム開発の基礎を学ぶのに最適です。

**特徴:**
- 基本的な弾幕システム
- 敵の生成と移動
- スコア機能
- ライフシステム
- レベルシステム

**学習ポイント:**
- Canvas APIの基本操作
- ゲームループの実装
- 衝突判定の基本
- 状態管理
- オブジェクトの生成と削除

### 2. パワーアップ版 (`game2_powerups`)
**難易度: 中級**

アイテムを集めてパワーアップ！連射速度が上がったり、弾が増えたりする戦略的なゲーム。

**特徴:**
- パワーアップアイテム（5種類）
  - 🔴 赤: 連射速度アップ
  - 🔵 青: マルチショット（3方向）
  - 🟢 緑: 移動速度アップ
  - 🟡 黄: スコア2倍
  - ⚪ 白: ライフ回復
- マルチショット対応
- タイマー機能付きパワーアップ
- アイテムドロップシステム

**学習ポイント:**
- アイテムシステムの実装
- タイマー管理
- 動的なゲーム要素の追加
- 状態の永続化
- パワーアップの効果管理

### 3. 物理エンジン版 (`game3_physics`)
**難易度: 上級**

リアルな物理演算で弾や敵が動く！重力と衝突を体験できる高度な実装。

**特徴:**
- 物理演算システム
- 重力システム
- 衝突応答（反発係数）
- 敵同士の衝突
- 摩擦システム
- 速度ベースの移動

**学習ポイント:**
- 物理エンジンの基礎
- 衝突検出アルゴリズム
- 衝突応答の実装
- 物理シミュレーション
- ベクトル演算

### 4. 特殊効果版 (`game4_special`)
**難易度: 最上級**

爆発エフェクト、パーティクル、特殊な弾！最もエキサイティングで複雑なバージョン。

**特徴:**
- 特殊弾（3種類）
  - 通常弾: 基本的な弾
  - 爆発弾: 範囲攻撃（敵を複数倒せる）
  - レーザー: 貫通攻撃（全敵に当たる）
- パーティクル効果
- 爆発システム
- 視覚効果の実装

**学習ポイント:**
- 特殊効果システム
- パーティクルシステム
- 視覚効果の実装
- 複雑なゲームロジック
- エフェクトのライフサイクル管理

## 操作方法

すべてのゲームで共通の操作方法:

- **マウス**: プレイヤーを動かす
- **矢印キー / A/Dキー**: プレイヤーを左右に動かす
- **スペースキー / クリック**: 弾を撃つ
- **スタートボタン**: ゲームを開始
- **一時停止ボタン**: ゲームを一時停止/再開

### 特殊効果版の追加操作

- **1キー**: 通常弾に切り替え
- **2キー**: 爆発弾に切り替え
- **3キー**: レーザーに切り替え

## 技術スタック

- **HTML5 Canvas**: ゲーム描画
- **Vanilla JavaScript**: ゲームロジック
- **CSS3**: スタイリング

外部ライブラリは使用していません。すべての機能を純粋なJavaScriptで実装しています。

### 3. 実装パターン

### 4. 拡張のヒント
各ゲームは独立しているので、自由に拡張や改造が可能です。

## 起動方法

### ローカルサーバーで起動

Python 3がインストールされている場合:

```bash
# Python 3の場合
python -m http.server 8000
```

または

```bash
# Python 2の場合
python -m SimpleHTTPServer 8000
```

Node.jsがインストールされている場合:

```bash
# http-serverを使用
npx http-server -p 8000
```

ブラウザで `http://localhost:8000` にアクセスしてください。
