# 2Dシューティングゲーム集

4つの異なるスタイルで楽しめる2Dシューティングゲームのコレクションです。それぞれ独自のシステムと魅力を持った4つのバージョンをお楽しみいただけます。

## 🎮 ゲーム紹介

### 1. クラシック版
**シンプルで爽快な基本プレイ**

初心者でも楽しめる、シンプルで分かりやすい2Dシューティングゲームです。敵を倒してスコアを稼ぎ、レベルアップを目指しましょう！

**主な特徴:**
- 直感的な操作で誰でも楽しめる
- 敵を倒すたびにスコアが増加
- レベルが上がるほど敵の出現速度がアップ
- ライフシステムで緊張感のあるゲームプレイ

### 2. パワーアップ版
**戦略的なアイテム収集が鍵！**

敵を倒すとアイテムがドロップ！5種類のパワーアップアイテムを駆使して、より高いスコアを目指しましょう。

**パワーアップアイテム:**
- 🔴 **連射速度アップ**: 弾を素早く撃てるようになる
- 🔵 **マルチショット**: 3方向に弾を撃てるようになる
- 🟢 **移動速度アップ**: プレイヤーの動きが速くなる
- 🟡 **スコア2倍**: 一定時間スコアが2倍になる
- ⚪ **ライフ回復**: ライフが1つ回復する

アイテムの効果は時間制限付き。タイミングよく使って、より高いスコアを目指してください！

### 3. 物理エンジン版
**リアルな物理演算が楽しめる！**

重力や衝突の影響を受ける、リアルな物理演算を体験できるバージョンです。弾や敵が物理法則に従って動くため、予測不能な展開が楽しめます。

**物理システム:**
- 重力の影響を受ける弾と敵
- 衝突時の反発効果
- 敵同士の衝突も発生
- 摩擦による減速

物理演算により、従来とは違った戦略的なプレイが可能です。

### 4. 特殊効果版
**華やかなエフェクトと特殊弾で大迫力！**

爆発エフェクト、パーティクル、3種類の特殊弾を駆使して、最もエキサイティングな戦いを楽しめます。

**特殊弾の種類:**
- **通常弾**: 基本的な弾（無制限）
- **爆発弾**: 範囲攻撃で複数の敵を一度に倒せる（限定数）
- **レーザー**: 画面全体を貫通する強力な攻撃（限定数）

敵を倒すと美しいパーティクルエフェクトが発生。視覚的にも楽しめる、最も派手なバージョンです！

## 🎯 操作方法

### 基本操作（全ゲーム共通）
- **マウス移動**: プレイヤーを動かす
- **矢印キー / A/Dキー**: プレイヤーを左右に動かす
- **スペースキー / マウスクリック**: 弾を撃つ
- **スタートボタン**: ゲームを開始
- **一時停止ボタン**: ゲームを一時停止/再開

### 特殊効果版の追加操作
- **1キー**: 通常弾に切り替え
- **2キー**: 爆発弾に切り替え
- **3キー**: レーザーに切り替え

## 🚀 プレイ方法

### オンラインでプレイ
GitHub Pagesで公開予定です。

### ローカルでプレイ
1. このリポジトリをクローンまたはダウンロード
2. ローカルサーバーを起動

**Python 3の場合:**
```bash
python -m http.server 8000
```

**Node.jsの場合:**
```bash
npx http-server -p 8000
```

3. ブラウザで `http://localhost:8000` にアクセス

## 💻 技術情報

このゲームは以下の技術で実装されています：
- **HTML5 Canvas**: ゲーム描画
- **Vanilla JavaScript**: ゲームロジック
- **CSS3**: スタイリング

外部ライブラリは一切使用していません。すべての機能を純粋なJavaScriptで実装しています。
