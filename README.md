# 2Dシューティングゲーム

パワーアップアイテムと特殊弾システムを統合した2Dシューティングゲームです。敵を倒してアイテムを集め、爆発弾やレーザーなどの特殊弾を駆使して高スコアを目指しましょう！

## 🎮 ゲームの特徴

### パワーアップアイテムシステム
敵を倒すと30%の確率でアイテムがドロップします。5種類のパワーアップアイテムを駆使して、より高いスコアを目指しましょう。

**パワーアップアイテム:**
- 🔴 **連射速度アップ**: 弾を素早く撃てるようになる（10秒間）
- 🔵 **マルチショット**: 3方向に弾を撃てるようになる（15秒間）
- 🟢 **移動速度アップ**: プレイヤーの動きが速くなる（10秒間）
- 🟡 **スコア2倍**: 一定時間スコアが2倍になる（20秒間）
- ⚪ **ライフ回復**: ライフが1つ回復する（最大5まで）

### 特殊弾システム
3種類の特殊弾を使い分けて、戦略的に敵を倒しましょう。

**特殊弾の種類:**
- **通常弾**: 基本的な弾（無制限）
- **爆発弾**: 範囲攻撃で複数の敵を一度に倒せる（初期3発、レベルアップで補充）
- **レーザー**: 画面全体を貫通する強力な攻撃（初期2発、レベルアップで補充）

### エフェクトシステム
- **パーティクルエフェクト**: 敵を倒すと美しいパーティクルが発生
- **爆発エフェクト**: 爆発弾を使用すると迫力のある爆発が発生

## 🎯 操作方法

### 基本操作
- **マウス移動**: プレイヤーを動かす
- **矢印キー / A/Dキー**: プレイヤーを左右に動かす
- **スペースキー / マウスクリック**: 弾を撃つ
- **スタートボタン**: ゲームを開始
- **リスタートボタン**: ゲームを再開
- **一時停止ボタン**: ゲームを一時停止/再開

### 特殊弾の切り替え
- **1キー**: 通常弾に切り替え
- **2キー**: 爆発弾に切り替え（残弾数がある場合のみ）
- **3キー**: レーザーに切り替え（残弾数がある場合のみ）

## 🎨 ゲームシステム

- **スコアシステム**: 敵を倒すと10ポイント獲得（スコア2倍パワーアップ中は20ポイント）
- **レベルシステム**: 500ポイントごとにレベルアップし、敵の出現速度が上がる
- **ライフシステム**: 初期ライフは3。敵に当たると1減り、0になるとゲームオーバー
- **アイテムドロップ**: 敵を倒すと30%の確率でパワーアップアイテムがドロップ
